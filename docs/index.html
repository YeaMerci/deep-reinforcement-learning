Mathematical Arsenal

The formal machinery implemented across the course — from Bellman optimality to the clipped policy-gradient surrogate and maximum-entropy RL.

Objective

expected discounted return
$$J(\pi_\theta)=\mathbb{E}_{\tau\sim\pi_\theta}\!\left[\sum_{t=0}^{\infty}\gamma^{t}R_t\right]$$

Bellman Optimality

dynamic programming
$$Q^{*}(s,a)=\mathbb{E}\!\Big[R_{t+1}+\gamma\max_{a'}Q^{*}(S_{t+1},a')\Big]$$

Q-Learning

off-policy TD control
$$Q(s,a)\leftarrow Q(s,a)+\alpha\big[r+\gamma\max_{a'}Q(s',a')-Q(s,a)\big]$$

DQN Loss

value function approximation
$$\mathcal{L}(\theta)=\mathbb{E}\Big[\big(r+\gamma\max_{a'}Q_{\bar\theta}(s',a')-Q_\theta(s,a)\big)^{2}\Big]$$

PPO · Clipped Surrogate

on-policy actor-critic
$$L^{\text{CLIP}}=\mathbb{E}_t\!\Big[\min\big(r_t\hat{A}_t,\;\text{clip}(r_t,1-\epsilon,1+\epsilon)\hat{A}_t\big)\Big]$$

Soft Actor-Critic

maximum-entropy RL
$$J(\pi)=\sum_t\mathbb{E}\Big[r(s_t,a_t)+\alpha\,\mathcal{H}\big(\pi(\cdot\mid s_t)\big)\Big]$$
